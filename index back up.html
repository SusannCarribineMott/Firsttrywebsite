THE FIRST THREE moves
The purpose of this website is to help people try a new game for the first time

When you are starting to play a game having a friend help you with the first couple of moves is super helpful. This website is intended to act like that friend. We who enjoy boardgames can help each other out.

How to use this webiste :)
Step 1. Watch a how to video, there are suggested links
Step 2. Set up your game.
Step 3. Follow along for the first three moves.

We chose to start with a few games we love that are awarded.
